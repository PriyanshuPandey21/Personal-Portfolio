## üöÄ Getting Started

To run this project locally, follow these steps:

1.  **Clone the repository**
    ```bash
    git clone https://github.com/PriyanshuPandey21/Personal-Portfolio.git
    cd Personal-Portfolio
    ```

2.  **Install Dependencies**
    ```bash
    yarn install
    # or
    npm install
    ```

3.  **Run the Project**
    ```bash
    yarn start
    # or
    npm start
    ```
